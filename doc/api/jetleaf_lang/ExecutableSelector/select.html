Selects a single executable from the given list based on all previously registered rules.

Behavior

  • Every rule must match at least one parameter of the executable.
  • If multiple executables match, the internal implementation determines how ties are resolved (commonly the first or most specific).
  • If none match, the implementation may throw, return null, or use fallback logic.
