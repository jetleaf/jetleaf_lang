ExecutableArgument resolve(
  1. Executable executable
)

Resolves the argument list for the provided executable.

This method examines the executable's parameters and attempts to match them with previously registered type-instance mappings. Each parameter may result in:

  • A positional argument inserted at the correct index
  • A named argument inserted using the parameter name

Unmatched parameters are left out of the result, meaning the caller must ensure the executable can still be invoked successfully.

Returns

An ExecutableArgument containing:

  • A list of positional argument values
  • A map of named parameters to values

These can be passed directly to Executable.newInstance().
