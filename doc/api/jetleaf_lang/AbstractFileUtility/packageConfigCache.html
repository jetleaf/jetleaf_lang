packageConfigCache property

List<PackageConfigEntry>? packageConfigCache
getter/setter pair

Cached list of parsed package configuration entries loaded from the active package_config.json.

This value is populated on-demand and reused for subsequent lookups to avoid repeated parsing of the configuration file.

null indicates that the configuration has not yet been loaded.
