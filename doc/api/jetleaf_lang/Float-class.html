Float

Float class

A wrapper class for single-precision floating point numbers.

This class wraps Dart's double type but represents it as a single-precision float, similar to Java's Float class.

Example usage:

Float a = Float(3.14);
Float b = Float.valueOf(2.71);
Float c = Float.parseFloat("1.618");

print(a.toString()); // "3.14"
print(a.compareTo(b)); // 1
