GarbageCollector

GarbageCollector class abstract final

A centralized, map-backed runtime garbage manager used by JetLeaf to track, evict, and periodically clean up cached or temporary objects.

GarbageCollector extends a specialized HashMap keyed by String, where each entry represents an internally managed garbage item.

Responsibilities

  • Track runtime objects by key
  • Enforce size limits and eviction policies
  • Perform periodic and aggressive cleanup
  • Provide typed and untyped access to tracked objects

Lifecycle

addGarbage → access → cleanup / eviction → removal

Cleanup may be triggered:

  • Automatically (periodic cleanup)
  • Aggressively (size or memory pressure)
  • Manually by the developer

Equality

Equality and hash code are derived from internal state via EqualsAndHashCode, ensuring deterministic comparisons.
