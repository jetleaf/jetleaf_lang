VersionRange

VersionRange class

Represents a simple semantic version range in Jetleaf.

A VersionRange defines an inclusive start version and an exclusive end version. It is used by the framework to determine whether a certain version (for example, Dart SDK version) satisfies a condition.

Usage Example:

final range = Range(
  start: Version(3, 0, 0),
  end: Version(4, 0, 0),
);

print(range.contains(Version(3, 2, 1))); // true
print(range.contains(Version(4, 0, 0))); // false
