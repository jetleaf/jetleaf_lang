AbstractSystemInterface

AbstractSystemInterface class abstract interface

Represents runtime system-level metadata for the JetLeaf framework.

The AbstractSystemInterface abstraction captures diagnostic and environmental information about the running application. This includes the entrypoint, compilation mode, launch command, IDE/debug status, and dependency/configuration statistics.

The framework can use this data to introspect the state of the runtime and provide detailed logs, telemetry, and diagnostics.

Example usage:

print(System.toSystemInfo());
