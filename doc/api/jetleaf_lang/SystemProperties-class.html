SystemProperties

SystemProperties class abstract interface

The SystemProperties interface provides a clean, method-based API for querying the runtime environment. Unlike direct property access, all system state is exposed via methods to enforce immutability and ensure consistent naming conventions.

Features

  • Runtime compilation mode detection
  • Execution environment analysis
  • IDE integration detection
  • Lifecycle & configuration metadata

Example

if (SystemEnvironment.isRunningAot()) {
  logger.info('Running in AOT mode');
}

if (SystemEnvironment.isIdeRunning()) {
  enableHotReload();
}
