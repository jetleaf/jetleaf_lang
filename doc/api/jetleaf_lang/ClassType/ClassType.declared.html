Creates a ClassType from a ClassDeclaration.

This constructor provides the most precise class reference by using complete declaration metadata. It's ideal for scenarios where exact type information is required.

Example

final declaration = ClassDeclaration.from(UserService);
final classType = ClassType.declared(declaration);
