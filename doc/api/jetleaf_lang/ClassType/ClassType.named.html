Creates a ClassType from a class name and package.

This constructor is convenient when you know the class name and its package location but don't have complete declaration metadata.

Example

final classType = ClassType.named(
  'UserService',
  'package:example/services'
);

@param name The simple name of the class @param package The package URI containing the class @param qualifiedName Optional fully qualified name @param pd Optional protection domain @param declaration Optional type declaration
