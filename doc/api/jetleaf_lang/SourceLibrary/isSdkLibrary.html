bool isSdkLibrary()

Indicates whether this library is part of the Dart SDK. isSdkLibrary acts as a classification signal that allows JetLeaf to distinguish between platform-provided libraries and user-controlled source code, ensuring correct behavior across analysis, reflection, and generation phases.

What This Means

Returns true when this SourceLibrary represents a library that is shipped with the Dart SDK itself, such as:

  • dart:core
  • dart:async
  • dart:collection
  • dart:io
  • dart:convert

and returns false for:

  • Application libraries
  • Third-party package libraries
  • Generated or workspace-local sources
