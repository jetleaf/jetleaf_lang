parallel abstract method

S parallel()

Returns an equivalent stream that is parallel.

May return itself, either because the stream was already parallel, or because the underlying stream state was modified to be parallel.

This is an intermediate operation.

Example

final stream = DartStream.of([1, 2, 3, 4, 5]);
final parallelStream = stream.parallel();
print(parallelStream.isParallel()); // true
